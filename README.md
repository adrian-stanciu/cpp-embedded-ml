### C++ image classifier with TensorFlow Lite on Raspberry Pi

#### Prerequisites
* hardware:
    * Raspberry Pi 4 model B
    * 8 GB microSD card
    * Raspberry Pi Camera module 3
* software:
    * 64-bit Raspberry Pi OS
    * g++ (with c++20 support)
    * make

#### Install
    sh install.sh

#### Compile
    make build

#### Run
    make run [IMAGE=<path-to-image>]

#### Clean up
    make clean

#### Uninstall
    sh uninstall.sh
